Exemplo 9 - CSS

O css é utilizado na estilização de elementos html. Para que possa aplicar a folha de estilos em cascata - css nos elementos html, você necessariamente poderá utilizar as regras css, que podem ser definidas por meio de seletores seguidos de {} e dentro dessas chaves, propriedades e valores.

Os seletores contidos nas regras CSS são padrões usados para definir quais elementos HTML receberão determinados estilos, eles podem ser de três tipos: Seletores de tipo, id e classe.

Seletores de tipo

Os seletores de tipo são aqueles que são representados pelas tags HTML, geralmente são aplicados a todas as tags HTML que se referenciam.

Esses seletores são descritos da seguinte forma: h1{propriedade:valor}, no caso do exemplo citado, o seletor de tipo é o h1, que representa uma tag de seção.

Seletores de ID

Os seletores de ID são aqueles que utilizam id(atributos) únicos de elementos HTML. Esses seletores são personalissimos, personalissimos pois são aplicados unicamente aos elementos HTML que tiverem o id específico.

Para criar um seletor de ID seguimos a seguinte regra: #NomeSeletor {propriedade:valor}

Seletores de Classe

Os seletores de classe são aqueles usados para aplicar o estilo a vários elementos HTML que possuírem o mesmo atributo class. Sua sintaxe é definida da seguintes maneira .NomeSeletor{prorpridade:valor}
